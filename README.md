# Aman Malhotra – Matrimonial Portfolio

A beautiful React-based matrimonial portfolio website.

## How to Run

1. Install dependencies:
   ```
   npm install
   ```

2. Start development server:
   ```
   npm start
   ```

3. Open http://localhost:3000 in your browser

## How to Build for Production

```
npm run build
```

The production files will be in the `build/` folder. Upload these to any web hosting service.

## Features
- Auto-rotating photo carousel on hero
- Full photo gallery with lightbox
- All personal, family, and career details
- WhatsApp contact button
- Mobile responsive design
- Smooth animations
